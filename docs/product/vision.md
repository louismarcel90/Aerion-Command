# Product Vision

AERION COMMAND is not a terminal game.

It is a deterministic tactical air combat and mission-assurance simulator.

## What It Proves

- simulation design
- deterministic runtime
- event-driven replay
- mission assurance
- fault handling
- evidence-grade output
- terminal UX craftsmanship

## Target Reviewer Reaction

A Staff/Principal reviewer should see:

- strong boundaries
- clear contracts
- testable engines
- replayable execution
- audit-ready evidence

A NASA/Government-minded reviewer should see:

- invariants
- degraded operations
- failure semantics
- traceability
- evidence chain
